# CTV Document Suite

## Запуск
```bash
pip install -r requirements.txt
python app.py
```

## Как пользоваться
1. Нажмите **Открыть…** — выберите xlsx-файл КП
2. Слева — список оборудования, справа — список работ
3. Нажмите **✏** на любой позиции чтобы отредактировать: название, описание, фото, цену, количество
4. **↑ ↓** — изменить порядок позиций
5. **✕** — удалить позицию
6. **＋ Добавить** — добавить новую позицию вручную
7. Нажмите **Создать КП** или **Создать Каталог**

## Структура
- `app.py` — главное приложение (запускать)
- `parser.py` — парсер xlsx
- `kp_pdf.py` — генератор КП
- `catalog_pdf.py` — генератор каталога
