# CTV Document Suite — Сборка и деплой

## Шаг 1 — Вписать URL сервера (одна строка)

Открыть `updater.py`, найти строку:

```python
EXE_URL = 'http://your-server.com/update/KPDesigner.exe'
```

Заменить на реальный URL — тот же путь, по которому старая программа
сейчас скачивает обновления. Новый exe будет лежать там же.

---

## Шаг 2 — Установить зависимости на Windows

```bat
pip install pyinstaller reportlab openpyxl Pillow pandas
```

Убедиться что в `C:\Windows\Fonts\` есть файлы `DejaVuSans*.ttf`.
Если нет — скачать пакет [dejavu-fonts](https://dejavu-fonts.github.io/)
и скопировать три файла в папку шрифтов.

---

## Шаг 3 — Собрать exe

```bat
cd ctv_suite
pyinstaller CTV_Suite.spec
```

Готовый файл: `dist\CTV_Document_Suite.exe`

Проверить вручную перед загрузкой:
- Открыть любой xlsx КП
- Создать PDF (КП и Каталог)
- Убедиться что всё работает

---

## Шаг 4 — Положить на сервер

Загрузить `dist\CTV_Document_Suite.exe` на VPS по тому же пути,
где сейчас лежит exe для обновлений.

Например через scp:
```bash
scp dist/CTV_Document_Suite.exe user@1.2.3.4:/var/www/html/update/KPDesigner.exe
```

---

## Что произойдёт дальше

Через 30 секунд после запуска (или через 20 минут если уже работают)
у всех пользователей появится диалог:

> «Найдена новая версия программы. Установить сейчас?»

Пользователь нажимает **Да** → прогресс-бар скачивает новый exe →
программа перезапускается. Всё автоматически, ничего вручную делать не нужно.

---

## Как выпускать следующие обновления

Просто загрузить новый `CTV_Document_Suite.exe` на сервер по тому же пути.
Программа сравнивает размер файла — если размер изменился, предлагает обновление.
